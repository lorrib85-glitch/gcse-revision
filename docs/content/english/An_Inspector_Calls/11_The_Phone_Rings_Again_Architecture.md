## 4. Build recommendations

1. **Do not make this a ghost mystery:** The Inspector's identity is interesting, but the exam value is ambiguity + structure + responsibility.

2. **Make cyclical structure visible:** Compare the first interruption with the final phone call. Same pattern, different emotional meaning.

3. **Centre "panic-stricken":** It proves Birling's lack of moral change in one stage direction.

4. **Use the curtain as Priestley's final tool:** The unresolved ending makes the audience carry the judgement out of the theatre.

5. **Keep Grade 9 extension optional:** Time theory can enrich, but it should not distract a 15-year-old from core structure.

6. **Design:** Use a dark dining-room phone, hard stage light, and fractured reflection of the opening scene. No horror ghosts, no detective red-string board, no jump-scare styling.
