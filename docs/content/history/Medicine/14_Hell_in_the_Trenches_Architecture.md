## Navigation spine (6 parts)

Every Medicine module must be built as six clear navigation parts. These titles should appear in the module journey/progress navigation so the student always knows where they are in the story.

1. **Medicine Under Fire** ŌĆö intro hook, surgery/public health recall and roadmap.
2. **Why the Western Front Was So Deadly** ŌĆö trenches, weapons, terrain, infection and evacuation problems.
3. **From Trench to Treatment** ŌĆö RAPs, ADS, CCS, base hospitals and the evacuation chain.
4. **New Problems, New Methods** ŌĆö wounds, shell shock, gas, blood transfusion, X-rays and triage.
5. **War Speeds Up Medicine** ŌĆö significance, limits and links to the whole Medicine course.
6. **Exam Prep: Historic Environment Mastery** ŌĆö source utility, follow-up question, features and exam practice.

## 2. Architecture checklist (tailored)

### Section 1 ŌĆö Intro, Recall & Roadmap
- **Purpose:** Recall surgery, infection and public health; frame the Western Front as the Historic Environment case study.
- **Proposed content for Episode 14:** PriorKnowledgeRecall, trench medical emergency hook, roadmap to conditions, evacuation, treatment and exam source skills.
- **Suggested components:** `CinematicRevealMoment`, `PriorKnowledgeRecall`, `WhatYouWillLearn`

### Section 2 ŌĆö Learning Chunk 1

- **Suggested components:** `VisualLearning`, `InteractiveHotspotImage`, `QuickRecallScreen`

## 4. Build recommendations

1. Treat this as both content and exam-technique module: the Historic Environment questions need deliberate practice.
2. Keep the evacuation chain visually clear and repeatedly retrieved.
3. Include source skills in Part 6, not scattered randomly through the teaching chunks.
4. Use links back to surgery, infection, public health and technology so this feels like the course finale.
