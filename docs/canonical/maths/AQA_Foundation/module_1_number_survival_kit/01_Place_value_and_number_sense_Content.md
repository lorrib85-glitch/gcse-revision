# Chapter 1: Place value and number sense — Content

## Series: AQA GCSE Maths Foundation — Module 1: Number survival kit

---

## Big question

Why can two numbers that use exactly the same digits mean completely different things — and how do examiners exploit that?

---

## AQA syllabus coverage

**Spec codes: N1, N2**

- Place value with integers: units, tens, hundreds, thousands, millions
- Place value with decimals: tenths, hundredths, thousandths
- Ordering positive integers (≤ 7 digits)
- Ordering negative integers
- Ordering decimals (including when decimal parts have different lengths)
- Ordering fractions by converting to a common denominator or decimal
- Using inequality symbols correctly: =, ≠, <, >, ≤, ≥
- Number lines: placing integers, decimals, and negative numbers accurately
- Understanding and use of place value for very large and very small numbers

---

## Key concepts and vocabulary

- **Place value:** the value of a digit determined by its position in a number (e.g. the 3 in 4,302 represents 300)
- **Digit:** one of the symbols 0–9
- **Integer:** a whole number (positive, negative or zero) — no decimal or fractional part
- **Decimal:** a number with a fractional part written after a decimal point (e.g. 4.75)
- **Inequality:** a statement showing that two values are not equal, or comparing their sizes
- **Number line:** a visual scale showing numbers in order of size; negative numbers extend to the left of zero
- **Ascending order:** smallest to largest
- **Descending order:** largest to smallest
- **Positive number:** greater than zero
- **Negative number:** less than zero
- **= (equals):** the two sides have the same value
- **≠ (not equal to):** the two sides have different values
- **< (less than):** e.g. 3 < 7 (read left to right: 3 is less than 7)
- **> (greater than):** e.g. 9 > 2 (9 is greater than 2)
- **≤ (less than or equal to):** includes the possibility of being equal
- **≥ (greater than or equal to):** includes the possibility of being equal

## Required formulae / techniques

No formulae. Core skills:

1. **Reading place value:** identify the value of any digit by its column (millions → units → tenths → thousandths)
2. **Ordering decimals:** align decimal points; compare column by column from the left
3. **Ordering negatives:** draw or imagine a number line; negative numbers with larger magnitude are *smaller* (e.g. −10 < −2)
4. **Writing inequalities:** choose the correct symbol; verify by reading the statement aloud ("is 0.3 less than 0.31?")
5. **Number line placement:** estimate position proportionally between known anchor points

## Likely misconceptions (from mark schemes and research)

- **MISCONCEPTION:** "0.35 is smaller than 0.3 because 35 is larger than 3." Correction: align decimal points — 0.30 vs 0.35; 0.35 > 0.30.
- **MISCONCEPTION:** "0.9 is smaller than 0.19 because 9 < 19." Correction: 0.9 = 0.90; comparing tenths column: 9 > 1, so 0.9 > 0.19.
- **MISCONCEPTION:** "−3 is greater than −1 because 3 is greater than 1." Correction: on a number line, −3 is to the left of −1; −3 < −1.
- **MISCONCEPTION:** "The number 4,302 has a zero because there is no value." Correction: zero is a placeholder — it holds the tens column so 3 stays in the hundreds column; removing it gives 432, a completely different number.
- **MISCONCEPTION:** Using ≤ and < interchangeably. Correction: ≤ includes the case where the two sides are equal; < does not.
- **MISCONCEPTION:** Placing 0.07 between 0.1 and 0.2 on a number line. Correction: 0.07 < 0.1; it belongs to the left of 0.1.

## Exam angles (from AQA past papers)

**How N1/N2 appears in Paper 1 (non-calculator):**

1. **Ordering task (1–2 marks):** "Write these numbers in order, smallest first: 0.3, 0.35, 0.09, 0.4" — tests decimal comparison
2. **Inequality notation (1 mark):** "Write the correct symbol (< or >) between 4.7 __ 4.69" — tests careful comparison at hundredths
3. **Number line placement (1 mark):** Mark the position of −3.5 on a number line labelled −5 to 0
4. **Context ordering (2 marks):** "Here are four temperatures: −8°C, 3°C, −1°C, 5°C. Write them in order, coldest first." — tests negative number understanding in context
5. **Embedded in multi-step problems:** place value errors compound in calculations — a student who writes 2.4 when they mean 24 loses all subsequent marks

**AQA wording patterns to recognise:**
- "Write in order of size, starting with the smallest"
- "Write a number that is between [X] and [Y]"
- "Use one of the symbols <, >, = to complete the statement"
- "Mark the number [X] on the number line"

**Mark scheme notes:**
- Ordering questions: one mark per correct position, or full marks for all correct; one error usually loses 1 mark (not all)
- ft (follow through): if an error is made in one part, subsequent parts often still score
- Inequality symbol: accept the correct symbol only — no ft for reversed symbols

## Maths in the wild — context bank

- **Temperatures:** comparing minimum overnight temperatures across a week (negative values common)
- **Sport:** comparing scores, rankings, times (athletics — lower is better; gymnastics — higher is better)
- **Finance:** bank balances with overdrafts (negative), price comparisons
- **Science data:** pH values, depths below sea level, altitudes
- **Shopping:** comparing unit prices (e.g. 1.8 litre vs 2 litre bottles)
- **Maps/coordinates:** understanding negative positions on a scale or grid
