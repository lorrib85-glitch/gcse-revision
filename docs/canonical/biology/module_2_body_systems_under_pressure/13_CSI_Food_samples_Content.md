# Chapter 13: CSI: Food samples — Content

## Series: Biology Module 2 — Body systems under pressure (AQA GCSE Combined Science Trilogy 8464, Biology Paper 1)

---

## Big question

How do scientists identify what is inside food — and how do you know if a test result is trustworthy?

---

## Syllabus coverage

- Food tests as a required practical
- Preparing food samples safely and consistently
- Benedict's test for reducing sugars: blue → green/yellow/orange/brick red precipitate if positive; heat required; glucose is a reducing sugar
- Iodine test for starch: orange-brown → blue-black if positive; no heating required
- Biuret test for protein: blue → lilac/purple if positive
- Lipid testing: Sudan III test or emulsion test, depending on school method
- Controls: using distilled water as a negative control and known food solutions as positive controls
- Variables: independent, dependent and control variables in a food test investigation
- Reliability and repeatability: repeating results, spotting anomalies and reducing contamination
- Presenting results in a table with clear headings
- Drawing valid conclusions from colour-change data

---

## Key concepts / vocabulary

- **Qualitative test:** a test that shows whether a substance is present or absent, usually through a visible colour change
- **Benedict's reagent:** blue solution used to test for reducing sugars; turns green/yellow/orange/brick red after heating if reducing sugar is present
- **Reducing sugar:** a sugar detected by Benedict's reagent, such as glucose
- **Iodine solution:** orange-brown solution used to test for starch; turns blue-black if starch is present
- **Biuret reagent:** reagent used to test for protein; turns lilac/purple if protein is present
- **Sudan III:** a red dye used to test for lipids; lipids form a red-stained oil layer
- **Emulsion test:** lipid test using ethanol and water; if lipid is present, a cloudy white emulsion forms
- **Positive result:** a colour change or observation showing the food molecule is present
- **Negative result:** no positive colour change; the tested molecule was not detected
- **Negative control:** a test with a known substance that should give a negative result, such as distilled water
- **Positive control:** a test with a known substance that should give a positive result, such as glucose solution for Benedict's test
- **Contamination:** when unwanted substances get into a sample and affect the result
- **Repeatability:** getting the same results when the same experiment is repeated using the same method

## Core explanation

Food tests are chemical detective work.

Each test uses a reagent that changes colour if a particular food molecule is present.

AQA does not just test whether students remember colours. It also tests method, safety, controls, reliability and whether conclusions match the evidence.

### Preparing a food sample

General method:

1. Crush or grind the food sample.
2. Add distilled water.
3. Stir or shake to dissolve soluble substances.
4. Filter if needed to remove solid pieces.
5. Use the liquid food sample for testing.

Use clean test tubes or spotting tiles to reduce contamination.

### Benedict's test for reducing sugars

Tests for:

- reducing sugars, such as glucose

Method:

1. Add food sample to a test tube.
2. Add Benedict's reagent.
3. Heat in a water bath for a few minutes.
4. Observe the colour change.

Positive result:

- blue → green/yellow/orange/brick red

Negative result:

- remains blue

Exam precision:

Benedict's test requires heating in a water bath. A stronger positive result tends towards brick red.

### Iodine test for starch

Tests for:

- starch

Method:

1. Place food sample in a spotting tile or test tube.
2. Add iodine solution.
3. Observe the colour change.

Positive result:

- orange-brown → blue-black

Negative result:

- remains orange-brown

Exam precision:

Iodine does not need heating.

### Biuret test for protein

Tests for:

- protein

Method:

1. Add food sample to a test tube.
2. Add Biuret reagent.
3. Shake gently.
4. Observe the colour change.

Positive result:

- blue → lilac/purple

Negative result:

- remains blue

Exam precision:

Biuret does not need heating.

### Lipid tests

Schools may teach Sudan III or the emulsion test. Know the method your teacher uses, but recognise both.

#### Sudan III test

Tests for:

- lipids

Method:

1. Add Sudan III stain to the food sample.
2. Shake gently.
3. Leave to stand.

Positive result:

- red-stained oil layer separates on top

#### Emulsion test

Tests for:

- lipids

Method:

1. Add ethanol to the food sample and shake.
2. Add water.
3. Observe.

Positive result:

- cloudy white emulsion

Negative result:

- solution remains clear

### Summary table

| Food molecule | Test/reagent | Positive result | Special condition |
|---|---|---|---|
| Reducing sugar | Benedict's reagent | green/yellow/orange/brick red | heat in water bath |
| Starch | Iodine solution | blue-black | no heat |
| Protein | Biuret reagent | lilac/purple | no heat |
| Lipid | Sudan III | red oil layer | leave to separate |
| Lipid | Emulsion test | cloudy white emulsion | ethanol, then water |

### Controls and trustworthiness

A control helps check that the test result is meaningful.

Negative control:

- use distilled water
- should give a negative result
- shows the reagent is not already contaminated

Positive control:

- use a known solution that should contain the molecule
- shows the test can detect that molecule

Reliability:

- repeat tests
- use clean equipment
- use the same sample volume and reagent volume where possible
- compare results with controls
- record results in a clear table

### Drawing conclusions

A conclusion must match the colour-change evidence.

Example:

If iodine turns blue-black, the sample contains starch.

Do not write:

"The food contains sugar" unless Benedict's test gave a positive result.

## Likely misconceptions

- MISCONCEPTION: "Benedict's test works on all sugars." Correction: Benedict's detects reducing sugars such as glucose.
- MISCONCEPTION: "A negative result means no food molecule is present at all." Correction: a negative result only means that particular molecule was not detected by that particular test.
- MISCONCEPTION: "Iodine tests for sugar." Correction: iodine tests for starch. Starch turns iodine blue-black.
- MISCONCEPTION: "The Biuret test uses heat." Correction: Biuret does not require heating. Benedict's is the food test that requires heating.
- MISCONCEPTION: "A white emulsion means protein is present." Correction: a white emulsion means lipid is present. Protein is detected by Biuret turning lilac/purple.
- MISCONCEPTION: "You only need to do each test once." Correction: repeating results improves reliability and helps identify anomalous results.
- MISCONCEPTION: "Colour change automatically proves the whole food is made of that molecule." Correction: it shows that molecule is present in the sample; the food may contain other molecules too.

## Suggested active learning checks

### Food test matching

Match:

- Benedict's reagent → reducing sugar → green/yellow/orange/brick red after heating
- iodine solution → starch → blue-black
- Biuret reagent → protein → lilac/purple
- Sudan III → lipid → red oil layer
- ethanol + water → lipid → cloudy white emulsion

### Method order: Benedict's test

Put in order:

- add food sample
- add Benedict's reagent
- place in water bath
- wait a few minutes
- observe colour change

### Spot the error

Student answer:

> I added iodine and heated it. It turned brick red, so starch was present.

Correction:

Brick red is the positive result for Benedict's test for reducing sugars after heating. Iodine tests for starch and turns blue-black if starch is present.

### Control judgement

Question:

Why use distilled water as a negative control?

Answer:

It should give a negative result, so it helps show that the reagent is not contaminated and that positive results are due to the food sample.

## Exam-style questions

### Question 1

Describe how to test a food sample for reducing sugar.  
3 marks

Mark points:

- add Benedict's reagent
- heat in a water bath
- positive result is green/yellow/orange/brick red

### Question 2

A student adds iodine solution to a food sample. It turns blue-black.

What conclusion can the student make?  
1 mark

Answer:

The food sample contains starch.

### Question 3

Describe how to test for protein and state the positive result.  
2 marks

Mark points:

- add Biuret reagent
- lilac/purple indicates protein

### Question 4

Explain why a negative control is useful in food testing.  
2 marks

Mark points:

- should not contain the food molecule being tested
- shows that any positive result is due to the sample / checks reagent contamination

### Question 5

A student says, "My sample did not turn blue-black with iodine, so it contains no food molecules."

Explain why this conclusion is not valid.  
2 marks

Mark points:

- iodine only tests for starch
- the sample may contain other molecules such as sugar, protein or lipid

### Question 6

A student tests four food samples. Describe how they should make the investigation more reliable.  
3 marks

Mark points:

- repeat tests
- use clean equipment
- use controls / compare with known positive and negative samples
- keep volumes/concentrations consistent
