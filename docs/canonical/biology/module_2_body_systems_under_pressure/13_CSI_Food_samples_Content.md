# Chapter 13: CSI: Food samples — Content

## Series: Biology Module 2 — Body systems under pressure (AQA GCSE Combined Science Trilogy 8464, Biology Paper 1)

---

## Big question

How do scientists identify what is inside food — and how do you know if a test result is trustworthy?

---

## Syllabus coverage

- Food tests as a required practical
- Benedict's test for reducing sugars: blue → brick red/orange precipitate if positive; heat required; glucose is a reducing sugar
- Iodine test for starch: orange/yellow → blue-black if positive; no heating required
- Biuret test for protein: blue → purple/lilac if positive; sodium hydroxide + copper sulfate solution
- Sudan III test (or emulsion test) for lipids: lipids stain red with Sudan III; emulsion test — add ethanol then water; white emulsion forms if lipid present
- Controls: using distilled water as a negative control
- Variables: independent, dependent, control variables in a food test investigation
- Reliability and repeatability: importance of repeating results
- Presenting results in a table with correct headings and units
- Drawing valid conclusions from colour change data

## Key concepts / vocabulary

- **Benedict's reagent:** blue solution used to test for reducing sugars; turns brick red/orange on heating if a reducing sugar is present
- **Reducing sugar:** a sugar that can donate electrons to reduce another compound; glucose, maltose and fructose are reducing sugars (sucrose is not)
- **Iodine solution:** orange/yellow solution used to test for starch; turns blue-black if starch is present
- **Biuret reagent:** sodium hydroxide + copper sulfate solution; turns purple/lilac in the presence of protein (detects peptide bonds)
- **Sudan III:** a red dye used to test for lipids; lipids stain red; used as an alternative to the emulsion test
- **Emulsion test:** lipid test using ethanol and water; if lipid is present, a white milky emulsion forms when water is added
- **Negative control:** a test run with a known substance that should give a negative result (e.g. distilled water); confirms the reagent is working correctly
- **Positive control:** a test run with a known substance that should give a positive result; confirms the test can detect the molecule
- **Independent variable:** the variable that is changed deliberately in an experiment
- **Dependent variable:** the variable that is measured or observed in an experiment
- **Control variables:** variables that are kept constant to ensure a fair test
- **Repeatability:** the ability to get the same results when the same experiment is repeated under the same conditions

## Likely misconceptions

- MISCONCEPTION: "Benedict's test works on all sugars." Correction: Benedict's only detects reducing sugars. Sucrose (table sugar) is a non-reducing sugar — it will not give a positive result unless first broken down by hydrolysis.
- MISCONCEPTION: "A negative result means no food molecule is present at all." Correction: a negative result only means that particular molecule was not detected at a detectable concentration. It does not mean the food contains nothing.
- MISCONCEPTION: "Iodine tests for sugar." Correction: iodine tests for starch. Starch turns iodine solution blue-black; sugars do not.
- MISCONCEPTION: "The Biuret test uses heat." Correction: the Biuret test does not require heating. Only Benedict's test requires heating in a water bath.
- MISCONCEPTION: "A white emulsion in the emulsion test means protein is present." Correction: a white emulsion means lipid is present. Protein is detected by Biuret reagent turning purple.
- MISCONCEPTION: "You only need to do each test once." Correction: repeating results increases reliability. A single result could be anomalous or due to contamination.
